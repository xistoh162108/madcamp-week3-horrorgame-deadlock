# DEADLOCK (Project: Madcamp Week 3)

**Deadlock** is a psychological horror puzzle game built with React, Vite, and Framer Motion.
Players assume the role of an operator trapped in a bio-containment facility, tasked with managing unstable systems while being hunted by an entity that lurks in the darkness.

## üéÆ Gameplay Features

*   **System Management**: Manage power load (`Heat`) while performing tasks to prevent a blackout.
*   **Surveillance**: Toggle between Monitor View (Coding/Terminal tasks) and Room View (Environment awareness).
*   **The Door**: The only barrier between you and the entity. Close it when you sense danger, but watch your power consumption.
*   **Diegetic UI**: Interactive terminal, code editor, and realistic CRT monitor effects.
*   **Jumpscares**: Dynamic audio-visual scares triggered by player actions and random events.

## üöÄ Getting Started

### Prerequisites
*   Node.js (v18+)
*   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/xistoh162108/madcamp-week3-horrorgame-deadlock.git
    cd madcamp-week3-horrorgame-deadlock
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open `http://localhost:5173` in your browser.
