# 💀 DEADLOCK (데드락) - 프로젝트 소개

> "Room 104의 관리자가 되어, 어둠 속에서 시스템을 복구하고 생존하세요."

**Deadlock**은 React와 Framer Motion으로 제작된 웹 기반 **심리적 공포 퍼즐 게임**입니다. 바이오 격리 시설의 오퍼레이터가 되어, 폐쇄적인 공간 압박과 청각적 공포 속에서 임무를 수행해야 합니다.

## 🎮 주요 기능 (Key Features)

### 1. 몰입형 인터페이스 (Diegetic UI)
게임 내 모든 UI는 실제 게임 세계의 장치(CRT 모니터, 터미널, 실제 문)처럼 작동합니다. 별도의 체력 바나 점수 판 없이, 모니터의 화면 찢어짐(Glitch)과 소리로 위기 상황을 전달합니다.

### 2. 멀티태스킹 서바이벌
플레이어는 두 가지 시점을 오가며 생존해야 합니다.
*   **Monitor View**: 코드를 입력하고 시스템 전력을 관리합니다. (업무)
*   **Room View**: 모니터를 끄고 주변을 살피며, 다가오는 괴물로부터 문을 방어합니다. (생존)

### 3. 청각적 공포 & 점프스케어
언제 튀어나올지 모르는 시각적 공포(Jumpscare)와 3D 사운드 효과가 결합되어 극강의 긴장감을 조성합니다.

## 🛠️ 기술 스택 (Tech Stack)

| 구분 | 기술 | 설명 |
| :--- | :--- | :--- |
| **Frontend** | React, TypeScript | 컴포넌트 기반 구조 설계 |
| **Build** | Vite | 빠른 개발 환경 및 빌드 최적화 |
| **Styling** | Tailwind CSS | 유틸리티 퍼스트 스타일링 |
| **Animation** | Framer Motion | 부드러운 UI 전환 및 공포 연출 |
| **State** | Zustand | 전역 상태 관리 (게임 로직) |
| **Audio** | Howler.js | 효과음 및 배경음 제어 |
